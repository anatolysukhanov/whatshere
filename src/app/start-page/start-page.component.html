A massive data set

We currently store over 12 million data records in our database. Currently only German addresses are available, but we work hard to extend the data set.

For the calculation we use map data and look at the environment for each individual address. This includes analyses of the transport infrastructure, the structural environment as well as the type and number of shops and service providers.

Powered by the latest technology

With the help of our large database we know exactly where businesses are and where they should be. We use machine learning to calculate the information you need to better assess the attractiveness of a location. We apply therefore latest technology from PostGis, Angular, JSON, Nvidia CUDA with xgBoost and many more.

About us

Whatshere.io has been started as a research project. We simply wanted to have more quantitative information about addresses. Whether this is for a home move, an evaluation of a neighbourhood if you want to buy a new house or just for fun.
